### 🚨 Stop deploying AI backdoors
Every third-party AI skill or MCP script you install is a potential entry point for hackers. AgentFend protects you against:

* **Prompt Injections:** Stop scripts from hijacking your agent's logic.
* **Data Exfiltration:** Prevent silent leaks of your API keys and environment variables.
* **Malicious Code:** Identify hidden scripts that execute unauthorized commands.

### ✨ How it works
No complex setup. No installation required.

1.  **Paste & Scan:** Input any AI skill URL or script.
2.  **Analyze:** Our engine runs static and behavioral audits in a secure sandbox.
3.  **Get Certified:** Receive a 0-100 Security Score and a detailed risk report.
